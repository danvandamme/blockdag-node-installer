BlockDAG Node Installer

Installs the full BlockDAG mining node stack on a Windows machine using Docker. Includes two redundant nodes, a stratum pool, PostgreSQL database, and a web dashboard.

Prerequisites

RequirementNotes
Windows 10 or 11 (64-bit) WSL 2 must be enabled. Docker Desktop will prompt you to enable it if needed.
Docker Desktop Required. The installer will start Docker automatically if it is installed but not running.
Python 3.8+ — optional Needed for the web dashboard. If Python is not found, the installer will note this and the dashboard shortcut will show instructions. Install Python then re-run the installer to enable it.
RAM: 8 GB minimum, 16 GB recommended The stack runs two full nodes, a pool, and a database concurrently.
Disk: 100 GB+ free space Chain data grows over time. An SSD is strongly recommended.

Quick Start

Double-click install-node-windows.bat, or run from a terminal:

.\install-node-windows

The installer will ask for your mining wallet address, then proceed automatically.

What the Installer Does

  1. Verifies that Docker is installed and starts it if needed
  2. Stops and removes any existing BlockDAG containers
  3. Loads the BlockDAG Docker images from the bundled .tar files
  4. Asks for your wallet address (the only prompt)
  5. Creates C:\blockdag node\ and copies all files into it
  6. Writes the .env configuration file with your wallet and default settings
  7. Creates the shortcuts folder and a desktop shortcut
  8. Starts the node stack automatically
  9. Launches the web dashboard and opens it in your browser (requires Python)

Desktop Shortcut

A single BlockDAG Node shortcut is placed on the desktop. It opens the C:\blockdag node\shortcuts\ folder which contains:

ScriptWhat it does
start-node.batStarts the full node stack
stop-node.batStops the node stack
node-status.batShows running containers and recent pool logs
node-logs.batStreams live logs from all containers (Ctrl+C to exit)
dashboard.batOpens the web dashboard, starting the server if needed

Web Dashboard

The dashboard runs at http://localhost:8088 and provides live monitoring and management:

  • Node sync status, block height, peers, and uptime
  • Pool hashrate, workers/miners with IP addresses, share counts
  • Payout history and balance information
  • Container health and resource usage
  • Peer manager with selective import
  • Live log viewer

Use the Config button in the toolbar to change pool settings (fee, difficulty, stratum port, password, wallet address) without editing files manually. Changes are applied to both .env files and the stack is restarted automatically.

Dashboard requires Python. Install Python 3.8+ and re-run the installer if the dashboard did not start. Tick "Add Python to PATH" during the Python installer.

Connecting Miners

Point your ASIC or miner software at the stratum port on this machine:

stratum+tcp://YOUR_SERVER_IP:3334

Default pool password: 1234 — change it in the dashboard Config window.

Everything lives under C:\blockdag node\:

shortcuts\Management batch scripts

.envMain configuration file

Changing Settings After Install

The easiest way is the Config button in the web dashboard — it saves to both .env files and restarts the stack automatically.

Alternatively, edit C:\blockdag node\.env directly, copy it to C:\blockdag node\asic-pool\.env, then restart using start-node.bat.

Pool private key (for automatic payouts): set POOL_PRIVATE_KEY manually in both .env files. Do not enter it into any web interface.

Chain Data Pre-Seeding (Optional)

If you have a chain data snapshot, copy its contents into C:\blockdag node\chain-data\node1\ and node2\ before starting the stack. This skips syncing from block 0 and can save many hours.
